1/ CSS UNITS (ĐƠN VỊ CSS): -bài trc: đơn vị em: kích thước phụ thuộc vào kích thước "px"của chính nó, sau đó đến "px"của thẻ cha, rồi mới đến "px trình duyệt=16px" -đơn vị rem: kích thước lấy từ font-size mặc định của trình duyệt=16px VD: font-size:3rem; //kích thước gấp 3 lần font chữ root element=48px -trong thực tế, rem đc sd rất nhiều vì nó cố định theo kích thước thẻ html mặc định là 16px, các thành phần khác của trang đều giữ nguyên tỉ lệ -ta có thể thay đổi kích thước mặc định và các thông số khác của thẻ html: ### file style.css: html { font-size:20px; //gọi thẻ html và thay đổi cỡ chữ mặc định cho nó } 2/QUY ĐỔI PX SANG REM: -ta có: 1rem=16px -mà khi tính toán ta cần quy đổi tle 1rem=10px => 10/16*100% = 62.5% => cần giảm font-size của HTML xuống 62.5% để 1rem=10px -tức là có những ptu có kích cỡ 40px,50px..(ko phải bội của 16px) nên khi đổi ra rem nó xấu,hơn nữa khi sửa ta phải sửa từng kích cỡ một mất tgian -ta sẽ sửa thẻ html thành 62.5%(10px) so với ban đầu 100%(=16px) Document
